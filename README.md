## How to start?

Initialize a new npm project:

```
npm init -y
```

Install Hardhat as a development dependency:

```
npm install --save-dev hardhat
```

Run the Hardhat initialization command:

```
npx hardhat init

```
```
npx hardhat compile
```

...Deploy all the needed contracts in scripts.

```
npm run start
```
